이 자료의 출처와 만듦새

누가, 어떻게 만들었고, 무엇을 참고했는지 투명하게 밝힙니다.

01제작 주체

이 학습 세션은 사내 학습동아리(CoP) AXIS — 폐쇄망 환경의 로컬 sLM(소형언어모델)·에이전트 활용을 연구하는 그룹 — 가 자체 제작한 자습 자료입니다. 특정 벤더나 외부 기관의 후원 없이, 연구회 학습 목적으로 작성했습니다.

02콘텐츠 작성 방식

본문의 설명·비유·퀴즈는 연구회가 직접 작성한 원문입니다. 외부 자료의 문장을 그대로 옮기지 않고, 개념을 이해한 뒤 우리 말과 우리 업무 맥락(폐쇄망, 공유 GPU 서버, 민원 처리 같은 실제 사례)에 맞춰 새로 설명했습니다.

내부 실증 수치에 대하여
일부 모듈의 실측 수치(예: 민원 100건 분류의 모델별 처리 시간, PII 비식별화 데모)는 연구회가 실제 업무 데이터를 대상으로 진행한 내부 사전 실증의 결과입니다. 외부에 공개된 학술 자료가 아니며, 방법론(데이터 구성·모델·측정 조건)은 연구회 내부 문서로 관리합니다.

03외부 자료 인용 원칙

  • 링크만 겁니다. 외부 글의 문장을 옮기지 않고, 한두 줄 요약과 함께 원문으로 안내합니다.
  • 성격을 밝힙니다. 공식 문서인지, 동료 심사 전 연구(프리프린트)인지, 개인 블로그·커뮤니티 글인지 표시합니다.
  • 신뢰도 주의를 답니다. 연구 단계 주장, 홍보성 글, 시범(PoC) 단계 사례에는 주의 문구를 함께 답니다.
  • 무료·공개 자료만 링크합니다. 로그인·결제가 필요한 자료는 걸지 않습니다.
  • 정기적으로 재확인합니다. 링크가 깨지거나 설명과 실제 내용이 달라지면 바로잡습니다. (2026-07 전수 감수 완료)

05진행 데이터 고지

퀴즈·자가진단 진행 상황은 여러분의 브라우저(localStorage)에만 저장되며 서버로 전송되지 않습니다. "진행 코드 복사" 버튼으로 내보낸 코드만, 여러분이 직접 연구회 실습 플랫폼에 등록할 수 있습니다.

연구회 타임라인 — 7월부터 12월까지

우리가 어디까지 왔고 다음에 무엇을 하는지 한눈에 봅니다. 각 달의 활동이 끝나면 답할 수 있어야 하는 레벨 질문이 함께 붙어 있습니다. 진행 상태 관리(완료 체크·일정 변경)는 회원 전용 실습 플랫폼의 타임라인에서 합니다 — 접속 방법은 모임에서 안내합니다.

7월 — 시작과 조준 Level 1·2

전원킥오프 미팅 — 하반기 운영 방식 합의
전원학습 사이트 모듈 0~2 자습 + 퀴즈 (진행 코드 등록)
전원업무 자동화 주제 제안서 제출 (사전 필터 2문항 통과)
전원공통 실증 태스크 2개 선정 + 역할 분담
리더공유 서버 1차 모델 서빙 + 성능 실측

8월 — 산정과 첫 호출 Level 3·4

세션산정 교육(90분): VRAM 손계산 → 계산기 검산 → 서버 실측 대조 → API 첫 호출
전원모듈 3~4 자습 (세션 전 선행)
전원공통 태스크 프로토타입 1차
전원중간발표(교류회) — "구형 GPU 24GB로 어디까지 되는가, 실측"

9월 — 측정의 달 Level 5

세션골든셋 스프린트(120분): 분포 읽기 → 합성 사례 생성 → 교차 검수 → v1 동결
전원모델 비교 평가 + 선정 사유서 작성

10월 — 완료를 의심하라 Level 6

세션검증 게이트 워크숍(120분): 완료 조건 도출 → 게이트 삽입 → 게이트 검증
전원PoC 실행 + 운영 리스크 기록

11월 — 조립과 제출 6레벨 통합

세션정리·보고 세션(120분): 성과 수치 확정(ROI) → 보고서 조립 → 발표 리허설
전원결과보고서 제출 (11/30)

12월 — 무대

전원우수 CoP 경진대회 — 6레벨 분담 발표
이 일정의 살아 있는 버전(진행 상태·완료 체크)은 회원 전용 실습 플랫폼(사내 접속)의 타임라인 메뉴에 있습니다. 이 페이지의 퀴즈 진행 코드를 플랫폼 대시보드에 등록하면 학습 진행도 계정에 함께 기록됩니다.
